Dans ce dossier les différentes releases du projet, avec les libs associées. 
Pour le bon fonctionnement du logiciel, suivez les étapes suivantes :
  1. créez un nouveau dossier qui contiendra votre logiciel et les bibliothèques qui lui sont associées
  2. mettez dans ce dossier la dernière version de ce logiciel ainsi que l'ensemble du contenu du sous-dossier 'libs'
  3. créez un nouveau dossier. il servira de dossier de sauvegarde.
  4. mettez-y le contenu du sous-dossier save.
  5. au lancement du programme une boîte de dialogue vous demandera le chemin vers ce dossier de sauvegarde. Indiquez-le. Cette boîte de 
     dialogue sera retirée dans une prochaine release.
C'est tout ! amusez-vous bien !
